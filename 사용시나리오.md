# 📘 SEO JIN PRECISION 생산관리 시스템 — 처음부터 끝까지

## 🎬 시나리오
**현대모비스**에서 **SDC80 CASE 부품 500개**를 주문 → 생산 → 검사 → 출하 → 통계 분석 까지 전체 흐름

---

## STEP 0 · 프로그램 실행

```
서진정밀 인트로 화면 표시
   ↓ Enter 키
메인 화면 진입 (admin 자동 로그인)
```

## STEP 1 · 🏢 고객사 등록

좌측 사이드바 **🏢 고객사 관리** 클릭

| 항목 | 입력값 |
|---|---|
| 코드 * | `C-002` |
| 회사명 * | `현대모비스` |
| 담당자 | 박과장 |
| 연락처 | 031-555-1234 |
| 이메일 | park@mobis.kr |

→ **🆕 고객사 등록** 클릭
→ 알림: "고객사 [현대모비스] 등록 완료!"

## STEP 2 · 📦 품목 등록

좌측 **📦 품목 관리** 클릭 (메뉴 미노출 시 시드 품목 사용)

| 항목 | 입력값 |
|---|---|
| 생산품명 * | `SDC80 CASE` |
| 규격 | Ø80 × 120 |
| 재질 | SUS304 |
| 단중 | 0.45 |
| 고객사 | 현대모비스 |

→ **저장**

---

## STEP 3 · 📋 수주 등록

좌측 **📋 수주 관리** 클릭

| 항목 | 입력값 |
|---|---|
| 주문번호 | `SJ-20260420-001` (자동) |
| 고객사 * | 현대모비스 |
| 생산품명 * | SDC80 CASE |
| 수량 * | `500` |
| 수주일 | 2026-04-20 |
| 납기일 | 2026-05-04 |
| 비고 | 긴급, 표면처리 무광 |

→ **🆕 수주 등록**
→ 상태: **접수**

---

## STEP 4 · 📅 생산계획

좌측 **📅 생산계획** 클릭

1. 목록에서 **SJ-20260420-001** 행 클릭
2. **작업지시 자동 생성** 클릭
3. → 일반CNC + 복합CNC **2건의 WO 자동 생성**
   - `WO-20260420-001-01` (일반CNC, 500ea)
   - `WO-20260420-001-02` (복합CNC, 500ea)
4. 수주 상태: 접수 → **진행중**

---

## STEP 5 · 🔧 작업지시 (설비/담당자 배정)

좌측 **🔧 작업지시** 클릭

### 5-1. 일반CNC 작업
- 목록에서 `WO-20260420-001-01` (공정: 일반CNC) 선택
- **설비 배정**: `LYNX2100M-1 - SDC80 CASE` (공정 일치 자동 필터링)
- **담당자**: 김생산
- → **배정** 클릭
- → **작업시작** → 상태: 대기 → **진행중**

### 5-2. 복합CNC 작업
- `WO-20260420-001-02` 선택
- 설비: `SR32J-1 - SEPERATOR` (또는 적합한 복합CNC 설비)
- 담당자: 이생산
- → **배정** → **작업시작**

---

좌측 **⚙️ 생산실적** 클릭

### 6-1. 4월 21일 (1일차)

| 작업일 | 양품 | 불량 |
|---|---|---|
| 2026-04-22 | 200 | 3 |

### 6-3. 4월 23일 (3일차) — 마무리

(복합CNC도 동일하게 진행)

좌측 **🚚 출하 관리** 클릭 (검사 합격된 주문만 표시)

## STEP 9 · 📈 보고서 출력

좌측 **📈 보고서 / 출력** 클릭

| 카드 | 클릭 결과 |
|---|---|
| **작업지시서** | 모든 WO 상세, 출력 미리보기 → 🖨 인쇄 (HP/Samsung 선택 가능) |
| **생산일보** | 일자별 양품/불량 합계, 불량률 자동 계산 |
| **거래명세서** | 출하 건별 A4 양식 (인수자 확인란 포함) |
| **설비별 가동** | 28대 설비별 누적 생산량 / 불량 |
| **📊 그래프 분석** | 6종 차트 (드롭다운 선택) → PNG 저장 가능 |

---

## STEP 10 · 📊 통계 분석 (일/월/년)

좌측 **📊 통계 (일/월/년)** 클릭

### 10-1. 📅 일별 통계
- **📅 일별 통계** 탭 클릭 → 최근 30일 자동 조회
- 구분 = 생산 → 🔍 조회

| 기간 | 작업건수 | 양품 | 불량 | 불량률(%) |
|---|---|---|---|---|
| 2026-04-21 | 1 | 180 | 5 | 2.70 |
| 2026-04-22 | 1 | 200 | 3 | 1.48 |
| 2026-04-23 | 1 | 120 | 2 | 1.64 |

→ **💾 CSV 저장** → `통계_생산_일별_2026-03-21_2026-04-20.csv`

### 10-2. 📆 월별 통계
- **📆 월별 통계** 탭 → 최근 12개월
- 구분 = 출하 → 🔍 조회 → 📊 그래프 저장 (PNG)

### 10-3. 🗓 연간 통계
- **🗓 연간 통계** 탭 → 최근 5년
- 구분 = 고객사별 → 🔍 조회
- 🖨 인쇄 → HP/Samsung 프린터 선택

## STEP 11 · 🏠 대시보드 확인

좌측 **🏠 대시보드** 클릭
- ⚠ 납기 임박 (7일 이내) 수주 목록
- 진행중 작업 현황
- 통계 카드: 총 수주 / 진행중 / 출하완료

---

## ✅ 완료된 데이터 흐름

```
고객사 등록 (현대모비스)
    ↓
품목 등록 (SDC80 CASE)
    ↓
수주 등록 (SJ-20260420-001 / 500ea)         [상태: 접수]
    ↓ 생산계획
작업지시 자동생성 (WO 2건)                   [상태: 진행중]
    ↓ 설비 배정 + 작업시작
생산실적 입력 (3일에 걸쳐 500ea 완료)
    ↓ 작업완료
품질검사 (합격)
    ↓
출하 등록 (SH-20260424-001 / 500ea)         [상태: 출하완료]
    ↓
보고서 출력 (작업지시서/생산일보/거래명세서)
    ↓
통계 분석 (일/월/년) → CSV/PNG/인쇄 저장
```

## 💡 팁 모음

| 상황 | 단축 방법 |
|---|---|
| 수주 수정 | 목록 행 클릭 → 자동 수정모드 → 값 변경 → ✏ 수주 수정 |
| 수주 삭제 | 목록 행 클릭 → 🗑 수주 삭제 (연관 작업/출하 함께 삭제 확인) |
| 고객사 수정/삭제 | 고객사 관리 → 행 클릭 → ✏ 수정 / 🗑 삭제 |
| 빠른 그래프 | 통계 → 탭 클릭 → 📊 그래프 저장 (즉시 PNG) |
| 프린터 변경 | 인쇄 다이얼로그에서 HP/Samsung 등 자유 선택 (A4 자동) |
| 설비 가동률 | 보고서 → 설비별 가동 → 28대 한눈에 |
| 페이지 스크롤 | 마우스 휠 (수직), Shift+휠 (수평), 우측/하단 스크롤바 |

## 📦 시스템 정보

- **프로그램명**: SEO JIN PRECISION 생산관리 시스템
- **회사**: 서진정밀
- **방식**: MTO (Make-to-Order, 주문생산)
- **공정**: 일반CNC + 복합CNC
- **번호 체계**:
  - 주문번호: `SJ-YYYYMMDD-NNN`
  - 작업지시: `WO-YYYYMMDD-NNN-SS`
  - 출하번호: `SH-YYYYMMDD-NNN`
- **기본 계정**: admin (자동 로그인, 별도 입력 불필요)
- **저장 형식**: SQLite (production.db)
