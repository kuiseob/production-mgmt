## 1. 전체 개요

### 시스템 구성 (3개 시스템)

```
┌─ 🏭 생산관리 시스템 (9개 메뉴) ─┐
│  대시보드 / 수주 / 생산계획       │
│  작업지시 / 생산실적 / 품질검사   │
│  출하 / 보고서 / 통계            │
└──────────────────────────────────┘

┌─ 🏬 외주관리 시스템 (5개 메뉴) ─┐
│  외주업체 / 외주품목 / 외주발주   │
│  입고/납품 관리 / 정산/지급 관리  │
└──────────────────────────────────┘

┌─ ⚙ 마스터 관리 (3개 메뉴) ─────┐
│  품목 관리 / 고객사 관리 / 설비   │
└──────────────────────────────────┘
```

### 시작 화면
- ⚙ **서진정밀** 인트로 → **Enter** 또는 클릭
- 메인 화면: 좌측 사이드바에 3개 시스템 (모두 접힘 상태)
- 시스템 클릭 → 펼침 + 메뉴 카드 그리드 표시

## 2. 실습 시나리오

### 가상 사례
**현대모비스**에서 **SDC80 CASE** 부품 **500개**를 주문 → 사내 **CNC 가공** + 외주 **열처리** + **도금** → 출하

### 사용할 데이터
| 분류 | 데이터 |
|---|---|
| 고객사 | **현대모비스** (CUS010) |
| 생산품 | **SDC80 CASE** |
| 수주 | 500EA, 납기 5/16 |
| 사내 공정 | LYNX2100M-1 (일반CNC), SR32J-1 (복합CNC) |
| 외주1 | **한국열처리** (V002) — 담금질 |
| 외주2 | **광명도금** (V003) — 경질크롬 도금 |

## 3. STEP 1 — 마스터 데이터 등록

먼저 **마스터 관리** 섹션 클릭 → 펼치기

### 3.1 🏢 고객사 관리

| 항목 | 입력값 |
|---|---|
| 코드 * | `CUS010` |
| 회사명 * | `현대모비스` |
| 담당자 | `박과장` |
| 연락처 | `031-555-1234` |
| 이메일 | `park@mobis.kr` |
| 우편번호 | `18581` |
| 도로명주소 | `경기 화성시 영천동 100` |
| 상세주소 | `현대모비스 R&D센터` |

→ **[고객사 등록]** (녹색)

### 3.2 📦 품목 관리

| 항목 | 입력값 |
|---|---|
| 생산품명 * | `SDC80 CASE` |
| 재질 | `SUS304` |
| 규격 | `Ø80 × 120` |
| 단중(kg) | `0.45` |
| 고객사 | `현대모비스` (드롭다운) |

## 4. STEP 2 — 생산관리 흐름

### 좌측 사이드바: **생산관리 시스템** 클릭 → 펼침

### 4.1 📋 수주 관리

| 항목 | 입력값 |
|---|---|
| 주문번호 | `SJ-20260502-001` (자동 생성) |
| 고객사 * | `현대모비스` |
| 생산품명 * | `SDC80 CASE` |
| 수량 * | `500` |
| 수주일 | `2026-05-02` |
| 납기일 | `2026-05-16` (자동: 14일 후) |
| 비고 | `긴급, 표면처리 무광` |

→ **[수주 등록]** → 상태: **접수**

### 4.2 📅 생산계획

1. 목록에서 `SJ-20260502-001` 클릭
2. **[작업지시 자동 생성]** (파란 버튼)
3. → 자동 생성됨:
   - `WO-20260502-001-01` (일반CNC, 500EA)
   - `WO-20260502-001-02` (복합CNC, 500EA)
4. 수주 상태: 접수 → **진행중**

### 4.3 🔧 작업지시

**A. 일반CNC 작업지시**
1. `WO-20260502-001-01` 행 클릭
2. **설비 배정**: `CNC-08 - LYNX2100M-1 (일반CNC)`
3. **담당자**: `김생산`
4. → **[배정 저장]** → **[작업 시작]** (상태: 진행중)

**B. 복합CNC 작업지시**
1. `WO-20260502-001-02` 행 클릭
2. **설비 배정**: `MCT-01 - SR32J-1 (복합CNC)`
3. **담당자**: `이생산`
4. → **[배정 저장]** → **[작업 시작]**

### 4.4 ⚙️ 생산실적 (3일치 입력)

**5월 3일 (1일차)**

|---|---|

| 작업일 | 2026-05-03 |
| 양품 수량 | `180` |
| 불량 수량 | `5` |
| 비고 | `1차 가공` |
→ **[실적 등록]**

**5월 4일 (2일차)**
| 작업일 | 양품 | 불량 |
|---|---|---|

**5월 5일 (3일차)**

|---|---|---|

→ **누적: 양품 500 / 불량 10** (계획 도달)
→ 작업지시 화면에서 **[작업 완료]**

### 4.5 🔍 품질검사

1. `SJ-20260502-001` 클릭

| 항목 | 값 |
|---|---|
| 검사구분 | `최종검사` |

## 7. 업무 흐름 다이어그램

```
[마스터 등록]                   [생산관리]
고객사 ─→ 품목 ─┐
                ├─→ 수주(접수) ─→ 생산계획(WO자동) ─→ 작업지시(진행중)
설비 ──────────┘                        ↓
                                    생산실적
                                        ↓
                                    품질검사(합격)
                                        ↓
                                    출하(출하완료)
                                        ↓
                                    📈 보고서 / 📊 통계


[외주관리] (생산과 병행)
외주업체 ─┐
          ├─→ 외주발주 ─→ 입고/납품 관리 ─→ 정산/지급 관리
외주품목 ─┘    (납기관리)   (납품률 자동)      (미지급 추적)
                                ↓
                          발주상태 자동 변경
                          (발주→진행→완료)
```

## 8. 자주 쓰는 단축 팁

### 화면 조작
| 단축키 | 동작 |
|---|---|
| Enter (인트로) | 메인 진입 |
| 마우스 휠 | 페이지 수직 스크롤 |
| Shift + 휠 | 페이지 수평 스크롤 |
| Cmd/Ctrl + Tab | 앱 전환 |
| Cmd/Ctrl + P (브라우저) | 인쇄 |

### 데이터 조작
| 동작 | 방법 |
|---|---|
| 신규 등록 | 폼 입력 → 녹색 [등록] 버튼 |
| 수정 | 목록 행 클릭 → 폼 자동 채워짐 → 주황 [수정] |
| 삭제 | 목록 행 클릭 → 빨강 [삭제] → 확인 |
| 사이드바 펼치기/접기 | 시스템 헤더 (▶/▼) 클릭 |

### 자동 계산 항목
| 메뉴 | 자동 계산 |
|---|---|
| 외주발주 | 합계 = 수량 × 단가 |
| 입고/납품 | 납품률 = 입고/발주 × 100% |
| 정산/지급 | 잔액 = 발주합계 - 지급액 |
| 생산실적 | 누적 양품/불량 자동 합계 |
| 통계 | 불량률 자동 계산 |
