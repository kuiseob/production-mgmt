## 1. 시작하기

### 1.1 프로그램 실행

**Mac**
```bash

```

- `SEOJIN_Production.exe` 더블클릭

## 5. 상세 사용법

### 5.1 🏠 대시보드

전체 업무 현황을 한눈에 확인합니다.

**표시 정보**
- ⚠ 납기 임박 수주 (7일 이내)
- 진행중 작업 현황
- 통계 카드 (총 수주 / 진행중 / 출하완료)

**조작**
- 단순 조회 화면 (편집 불가)
- 다른 메뉴로 이동해서 작업

---

### 5.2 📋 수주 관리

신규 주문을 등록하고 수정/삭제합니다.

**입력 항목**
| 항목 | 필수 | 설명 |
|---|---|---|
| 주문번호 | ✓ | 자동 생성 (SJ-YYYYMMDD-NNN) — "갱신" 버튼으로 새 번호 |
| 고객사 | ✓ | 드롭다운에서 선택 |
| 생산품명 | ✓ | 드롭다운에서 선택 |
| 수량 | ✓ | 정수 |
| 수주일 | ✓ | 기본값: 오늘 |
| 납기일 | ✓ | 기본값: 14일 후 |
| 비고 | - | 자유 입력 |

**3개 액션 버튼**
| 버튼 | 색상 | 동작 |
|---|---|---|
| 🟢 **수주 등록** | 녹색 | 신규 주문 INSERT |
| 🟠 **수주 수정** | 주황 | 선택한 주문 정보 변경 |
| 🔴 **수주 삭제** | 빨강 | 주문 + 연관 작업/검사/출하 함께 삭제 |

**수정 방법**
1. 하단 목록에서 행 클릭
2. 폼이 자동으로 데이터 채워짐
3. 값 변경 후 **수주 수정** 클릭

**삭제 방법**
1. 목록 행 클릭 후 **수주 삭제** 클릭
2. 연관 데이터 건수 표시됨 (예: "작업지시 2건 / 출하 1건 함께 삭제")
3. 확인 → 모두 삭제

---

### 5.3 📅 생산계획

수주를 작업지시(WO)로 변환합니다.

**조작**
1. 목록에서 수주 한 건 선택
2. **작업지시 자동 생성** 버튼 클릭 (파란색)
3. → 일반CNC + 복합CNC 2개의 WO가 자동 생성됨
4. 수주 상태: 접수 → **진행중**

**자동 생성 규칙**
- 한 수주당 공정 수만큼 WO 생성 (현재 2개: 일반CNC, 복합CNC)
- 각 WO는 같은 수량으로 분할
- WO 번호: `WO-YYYYMMDD-NNN-SS` (SS는 순번)

---

### 5.4 🔧 작업지시

설비/담당자를 배정하고 작업 진행을 관리합니다.

**입력 항목**
| 항목 | 설명 |
|---|---|
| 설비 배정 | 공정에 맞는 설비만 자동 필터링 (일반CNC ↔ 복합CNC) |
| 담당자 | 작업자 선택 |

**3개 액션 버튼 (한 줄)**
| 버튼 | 색상 | 동작 |
|---|---|---|
| 🟢 **배정 저장** | 녹색 | 설비/담당자 정보 저장 |
| 🔵 **작업 시작** | 파랑 | 상태: 대기 → **진행중** (시작 시각 기록) |
| 🟠 **작업 완료** | 주황 | 상태: 진행중 → **완료** (종료 시각 기록) |

**조작 순서**
1. 목록에서 작업 선택
2. 설비/담당자 선택 → **배정 저장**
3. 실제 작업 시작 시 → **작업 시작**
4. 작업 완료 시 → **작업 완료**

---

### 5.5 ⚙️ 생산실적

일자별 생산수량을 입력합니다.

**입력 항목**
| 항목 | 필수 | 설명 |
|---|---|---|
| 작업일 | ✓ | 기본값: 오늘 |
| 양품 수량 | ✓ | 정수 |
| 불량 수량 | - | 기본값: 0 |
| 비고 | - | 자유 입력 |

**액션 버튼**
- 🟢 **실적 등록** (녹색)

**조작**
1. 목록에서 진행중 작업 선택
2. 작업일/양품/불량 입력
3. **실적 등록** 클릭
4. 누적 실적이 계획 수량 도달 시, 작업지시 화면에서 **작업 완료** 처리

**불량률 자동 계산**
- 보고서/통계에서 자동으로 (불량/(양품+불량))×100 계산

### 5.6 🔍 품질검사

**액션 버튼**

### 5.8 📈 보고서 / 출력

5종 보고서를 미리보기/인쇄합니다.

**5개 카드** (보라색 "미리보기 / 인쇄" 버튼)

| 보고서 | 내용 |
|---|---|
| **작업지시서** | 모든 WO 상세 (작업번호/품명/공정/설비/계획/실적) |
| **생산일보** | 일자별 양품/불량 합계, 불량률 자동 계산 |
| **거래명세서** | 출하 건별 A4 양식 (인수자 확인란 포함) |
| **설비별 가동** | 28대 설비별 누적 생산량/불량 |
| **그래프 분석** | 6종 차트 (드롭다운 선택) |

**그래프 6종**
1. 일별 생산량 추이 (양품/불량 적층 막대)
2. 공정별 생산 비중 (파이)
3. 설비별 생산량 TOP 10 (가로 막대)
4. 일별 불량률 추이 (라인 + 3% 관리한계선)
5. 수주 상태 현황 (막대)
6. 고객사별 누적 수주량 TOP 10

**인쇄 흐름**
1. 카드 → "미리보기 / 인쇄" 클릭
2. 브라우저에서 HTML 미리보기 자동 열림 (A4 자동 설정)
3. 상단의 🖨 인쇄 버튼 또는 Ctrl+P
4. 프린터 선택 (HP/Samsung 등)
5. 인쇄

### 5.9 📊 통계 (일/월/년)

일별/월별/연간 집계를 자동 생성합니다.

**3개 탭 버튼** (상단)
| 탭 | 색상 | 자동 기간 |
|---|---|---|
| 📅 **일별 통계** | 파랑 | 최근 30일 |
| 📆 **월별 통계** | 초록 | 최근 12개월 |
| 🗓 **연간 통계** | 보라 | 최근 5년 |

**구분 선택 (드롭다운)**
- 생산 / 품질 / 출하 / 설비별 / 고객사별

**요약 카드 4종 (자동 집계)**
- 총 행 / 합계 / 불량합계 / 불량률(%)

**저장 액션 버튼 4종**
| 버튼 | 색상 | 동작 |
|---|---|---|
| 🔵 **조회** | 파랑 | 테이블 + 요약 갱신 |
| 🟣 **CSV 저장** | 보라 | Excel 호환 (UTF-8 BOM, 메타 포함) |
| 🟣 **인쇄** | 보라 | 프린터 출력 |
| 🟣 **그래프 저장** | 보라 | PNG 파일로 저장 |

**CSV 파일 구조 예시**
```
# SEO JIN PRECISION CO. - 통계 보고서
# 구분: 생산 / 기간: 일별 / 2026-03-26 ~ 2026-04-25 / 생성: 2026-04-25 16:30

기간,작업건수,양품,불량,불량률(%)
2026-04-21,1,180,5,2.70
2026-04-22,1,200,3,1.48
...
총 행,4
합계,500
불량합계,10
불량률,1.96 %
```

### 5.10 📦 품목 관리

부품 마스터를 등록합니다.

**입력 항목**

## 10. 자주 묻는 질문 (FAQ)

### Q1. 로그인이 필요한가요?
A. 아니요. 별도의 로그인 절차 없이 자동으로 관리자 권한으로 진입합니다.

### Q2. 한글 입력 후 저장 버튼을 눌러도 "필수 입력" 오류가 나요.
A. 한글 IME 미커밋 상태일 수 있습니다. **Tab 키나 Enter 키**로 입력을 확정한 후 버튼을 눌러주세요. 최신 버전은 자동 커밋되도록 처리되어 있습니다.

### Q3. 수주를 잘못 등록했어요. 어떻게 수정/삭제하나요?
A.
- **수정**: 목록 행 클릭 → 폼 자동 채워짐 → 값 변경 → **수주 수정** 버튼
- **삭제**: 목록 행 클릭 → **수주 삭제** 버튼 → 확인

### Q4. 같은 코드의 고객사를 다시 등록할 수 없어요.
A. 한 번 삭제(소프트 삭제)된 코드를 다시 등록하면 **자동으로 복원/갱신 옵션**이 제공됩니다. "복원" 선택 시 새 정보로 갱신됩니다.

### Q5. 작업지시는 어떻게 만드나요?
A.
1. **수주 관리**에서 주문 등록
2. **생산계획**에서 해당 주문 선택 → **작업지시 자동 생성** 클릭
3. → 일반CNC + 복합CNC 2개 WO가 자동 생성됨

### Q6. 그래프를 PNG로 저장하고 싶어요.
A. **통계 (일/월/년)** 메뉴 → 탭 선택 → **그래프 저장** 버튼 → 파일 위치 지정

### Q7. 데이터를 Excel에서 열고 싶어요.
A. **통계** 메뉴에서 **CSV 저장** 버튼 → 생성된 CSV 파일을 Excel에서 더블클릭

### Q8. 사이드바 메뉴가 잘 안 보여요.
A. 글씨 크기를 조정할 수 있습니다. 코드의 사이드바 폰트 설정을 변경하거나 개발자에게 요청하세요.

### Q9. 프린터에서 인쇄가 안 돼요.
A.
1. Windows에서 프린터가 정상 설치되어 있는지 확인
2. 인쇄 다이얼로그에서 올바른 프린터 선택
3. A4 용지 사용 권장

### Q10. DB 파일을 백업하고 싶어요.
A. 프로그램 종료 후 `production.db` 파일을 다른 위치로 복사하면 됩니다.

## 11. 문제 해결 (Troubleshooting)

### 11.1 프로그램이 실행되지 않을 때

**Mac**
```bash
# Python 3 확인
/opt/homebrew/bin/python3.13 --version

# 직접 실행
/opt/homebrew/bin/python3.13 /Users/kuiseob/production-mgmt/production_app.py
```

**Windows**
- EXE 우클릭 → 속성 → 차단 해제
- SmartScreen 추가 정보 → 실행

### 11.2 데이터베이스 오류 (DB locked)

- 동일한 DB 파일을 여러 프로그램에서 열고 있을 수 있습니다.
- 모든 프로그램 종료 → 재실행

### 11.3 화면이 깨져 보일 때

- 윈도우 크기 조정 (최소: 1100×680)
- 프로그램 종료 → 재실행

### 11.4 그래프가 표시되지 않을 때 (Mac)

- matplotlib 설치 필요:
  ```bash
  /opt/homebrew/bin/python3.13 -m pip install matplotlib
  ```
- 또는 venv 사용:
  ```bash
  /opt/homebrew/bin/python3.13 -m venv ~/prodenv
  ~/prodenv/bin/pip install matplotlib
  ~/prodenv/bin/python /Users/kuiseob/production-mgmt/production_app.py
  ```

### 11.5 프린터 선택 다이얼로그가 안 뜸 (Mac)

- macOS는 직접 프린터 선택 기능 없음 (Windows 전용)
- 브라우저 인쇄 다이얼로그(Ctrl+P)에서 선택 가능

### 11.6 한글이 깨질 때

- 폰트 "Malgun Gothic" 설치 확인 (Windows 기본 포함)
- Mac은 자동으로 시스템 한글 폰트 사용
