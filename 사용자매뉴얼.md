## 1. 시스템 개요

### 1.1 프로그램 목적
**서진정밀**은 기계부품을 **수주생산(MTO)** 방식으로 제조하는 정밀가공 전문 업체입니다.
이 시스템은 다음 업무를 통합 관리합니다:

- 📋 고객 **수주 등록/수정/삭제**
- 📅 **생산계획** (수주 → 작업지시 자동 변환)
- 🔧 공정별 **작업지시** (28대 설비 배정 + 수정/삭제)
- ⚙️ 일자별 **생산실적** 등록/수정/삭제
- 🔍 **품질검사** 합격/불합격 판정
- 🚚 **출하 관리** + 거래명세서
- 📈 5종 **보고서** + 6종 **그래프**
- 📊 일/월/년 **통계** 분석 + CSV/PNG/인쇄
- 📆 **일자별 이력** 통합 조회 ⭐ NEW
- 🏢 거래처 / 📦 품목 / 🏭 설비 마스터

### 1.2 핵심 특징

| 항목 | 내용 |
|---|---|
| **업무 방식** | MTO (Make-to-Order) |
| **공정** | 일반CNC, 복합CNC (2종) |
| **보유 설비** | 일반CNC 13대 + 복합CNC 15대 = **28대** |
| **데이터 저장** | SQLite (production.db, 단일 파일, 영속) |
| **로그인** | 없음 (자동 admin 권한) |
| **GUI** | Python tkinter (Mac/Windows 호환) |
| **그래프** | matplotlib (한글 자동 폰트) |
| **프린터** | HP, Samsung 등 설치된 모든 프린터 |
| **단축키** | Esc (초기화), Tab (다음 입력), Enter (확정) |

### 1.3 v3.0 새로운 기능 (2026-04-29)

⭐ **데이터 영속성 보장**
- Windows EXE 종료 후 재시작해도 데이터 유지
- PyInstaller frozen 환경 자동 감지

⭐ **📆 일자별 이력 메뉴 (신규)**
- 8종 데이터를 등록 일시 기준으로 통합 조회
- 종류별 색상 구분 + 요약 카드

⭐ **DB → PDF 백업 스크립트**
- `db_to_pdf.py` 더블클릭 → 전체 DB를 A4 PDF로

⭐ **모든 페이지 등록/수정/삭제 통일**
- 색상 일치 (녹/주/빨)
- 행 클릭 → 자동 수정 모드
- Esc 키 = 초기화 (모든 페이지)

⭐ **macOS Tk Aqua 호환성**
- IME 미커밋 자동 처리

## 2. 설치 및 실행

### 2.1 시스템 요구사항
- **OS**: Windows 10/11, macOS 12+, Linux Ubuntu 22+
- **메모리**: 최소 4GB, 권장 8GB
- **저장공간**: 100MB
- **화면**: 최소 1100×680, 권장 1340×820

### 2.2 첫 실행

**Windows**
```cmd
1. https://github.com/kuiseob/production-mgmt/releases 접속
2. SEOJIN_Production.exe 다운로드
3. 파일 우클릭 → 속성 → "차단 해제" 체크 → 확인
4. 더블클릭 실행
```

**macOS**
```bash
git clone https://github.com/kuiseob/production-mgmt.git ~/production-mgmt
brew install python@3.13 uv
uv venv --python 3.12 ~/.prod_venv
~/.prod_venv/bin/pip install matplotlib
~/.prod_venv/bin/python ~/production-mgmt/production_app.py
```

## 6. 상세 기능 설명 (메뉴별)

### 6.1 🏠 대시보드

**표시 정보**

- 진행중 작업 현황 — 노란 강조

**조작**: 단순 조회 (편집 불가)

---

### 6.2 📋 수주 관리

**입력 항목**

| 항목 | 필수 | 형식 |
|---|---|---|
| 주문번호 | ✓ | 자동 생성 (수정 가능): `SJ-YYYYMMDD-NNN` |
| 고객사 | ✓ | 드롭다운 |
| 생산품명 | ✓ | 드롭다운 |
| 수량 | ✓ | 정수 |
| 수주일 | ✓ | YYYY-MM-DD (자동 정규화: 20260429 → 2026-04-29) |
| 납기일 | ✓ | YYYY-MM-DD |
| 비고 | - | 자유 입력 |

**3개 액션 버튼**

| 버튼 | 색상 | 동작 |
|---|---|---|
| 🟢 **수주 등록** | 녹색 | 신규 INSERT (중복 검사) |
| 🟠 **수주 수정** | 주황 | 선택한 주문 UPDATE |
| 🔴 **수주 삭제** | 빨강 | 주문 + 연관 데이터 삭제 |
| ⚫ **초기화** | 회색 | 폼 비우기 (Esc 키) |

**조작 흐름**

A. **신규**: 폼 입력 → 수주 등록
B. **수정**: 목록 행 클릭 → 폼 자동 채워짐 → 값 변경 → 수주 수정
C. **삭제**: 행 클릭 → 수주 삭제 → 연관 데이터 건수 확인 → 확인

---

### 6.3 📅 생산계획

**동작**: 한 수주 → 일반CNC + 복합CNC **2개 WO 자동 생성**

**조작**
1. 목록에서 수주 선택
2. **작업지시 자동 생성** 클릭 (파란색)
3. → WO 2건 생성 (`WO-YYYYMMDD-NNN-01`, `-02`)
4. 수주 상태: 접수 → **진행중**

---

### 6.4 🔧 작업지시

**입력 항목**

| 항목 | 설명 |
|---|---|
| 설비 배정 | **28개 설비 모두** 표시 (일반/복합 자유 선택) |
| 담당자 | 작업자 |
| 계획수량 | 수정 가능 |

**5개 액션 버튼 (한 줄)**

| 버튼 | 색상 | 동작 |
|---|---|---|
| 🟢 **배정 저장** | 녹색 | 설비/담당자 저장 |
| 🔵 **작업 시작** | 파랑 | 상태 → 진행중 |
| 🟠 **작업 완료** | 주황 | 상태 → 완료 |
| 🟠 **WO 수정** | 주황 | 계획수량 변경 |
| 🔴 **WO 삭제** | 빨강 | WO + 생산실적 함께 삭제 |
| ⚫ 초기화 | 회색 | 폼 비우기 |

---

### 6.5 ⚙️ 생산실적

**3단 화면**: WO 선택 → 실적 이력 → 입력 폼

**입력**
- 작업일 / 양품 수량 / 불량 수량 / 비고

**3개 버튼**: 실적 등록 / 실적 수정 / 실적 삭제 + 초기화

**자동 처리**
- 등록/수정/삭제 시 WO의 누적 실적 자동 재계산
- 행 클릭 → 자동 수정 모드 (모드 라벨이 주황색으로 변경)

---

### 6.6 🔍 품질검사

**입력**
- 검사구분 (자주/최종) / 검사일 / 샘플수
- **결과** (라디오: 합격 / 불합격)
- 불량수 / 불량사유 / 비고

**3개 버튼**: 검사 결과 저장 / 검사 수정 / 검사 삭제 + 초기화

## 16. 문제 해결

### 16.1 프로그램이 실행되지 않음

**Windows**
- EXE 우클릭 → 속성 → 차단 해제
- SmartScreen 추가 정보 → 실행
- 백신 프로그램 격리 폴더 확인

**macOS**
```bash
/opt/homebrew/bin/python3.13 --version
~/.prod_venv/bin/python ~/production-mgmt/production_app.py
```

### 16.2 데이터베이스 오류 (DB locked)
- 다른 인스턴스 종료 → 재실행

### 16.3 그래프 표시 안 됨
matplotlib 재설치:
```bash
~/.prod_venv/bin/pip install --upgrade matplotlib
```

### 16.4 입력값이 저장되지 않음
- IME 상태 확인
- Tab/Enter로 입력 확정 후 저장

### 16.5 EXE 파일이 손상되었다고 나옴
- 다시 다운로드
- 안티바이러스 일시 중지

### 16.6 Windows에서 데이터가 사라짐
- v3.0 이전 버전: PyInstaller 임시 폴더 문제 → v3.0 이상으로 업데이트
- v3.0+: EXE를 다른 폴더로 옮길 때 production.db도 함께 이동
